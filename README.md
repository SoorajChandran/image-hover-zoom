# React component for zoom on hover ( e-commerce websites )

Simple component for zoom on hover. [Live Demo](https://soorajchandran.github.io/image-hover-zoom/)


Clone the repo

```sh
$ npm install
$ npm start
```

### Linting


```sh
$ yarn run lint
```


### watch scss change

```sh
$ npm run watch-css
```
